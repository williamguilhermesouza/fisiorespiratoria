## How to install:

Clone the git repository:

> git clone https://github.com/williamguilhermesouza/fisiorespiratoria.git

Enter the repository:

> cd fisiorespiratoria

Use expo to install the dependencies:

> expo install

Start the bundler with expo:

> expo start
